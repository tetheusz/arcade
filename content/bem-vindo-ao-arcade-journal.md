# Começo do arquivo

Este espaço nasceu para virar uma base viva de conteúdo sobre o ecossistema Arc para builders brasileiros.

## O que você vai encontrar aqui

- resumos de docs em PT-BR
- guias práticos de onboarding
- notas de pesquisa sobre builders e infraestrutura
- ideias de apps e tools
- recaps semanais do grind

## Como este arquivo vai crescer

A proposta do Arcade é simples:

1. estudar um conceito
2. transformar isso em conteúdo público
3. construir algo útil em cima
4. repetir até ganhar consistência

Este primeiro mês é sobre contexto, repertório e base. Depois, a ideia é evoluir para analytics, mini apps e ativos mais profundos para o ecossistema.
